# Lee Sangyoon Portfolio

정보공학 전공 과정에서 수행한 시스템 프로그래밍, 네트워크 프로그래밍, 컴파일러, GUI 및 응용 프로그램 개발 프로젝트를 정리한 포트폴리오입니다.

C / Linux 환경에서 TCP/IP Socket 기반 채팅 서버와 UNIX Shell을 구현하며 소켓 통신, I/O Multiplexing, 프로세스 실행, 명령어 파싱, 자료구조 기반 상태 관리 경험을 쌓았습니다. 또한 Java 기반 Pascal-like Compiler 프로젝트를 통해 문법 분석, 의미 분석, 코드 생성, 스코프 관리 등 복잡한 프로그램 구조를 단계적으로 구현하고 디버깅하는 경험을 했습니다.

## Main Projects

1. [Pascal-like Compiler](./projects/compiler)  
   Java 기반 Pascal 계열 언어 컴파일러 프로젝트입니다. 문법 분석, 의미 분석, 코드 생성, 스코프 관리, CASL II 코드 생성을 구현했습니다.

3. [Mini UNIX Shell](./projects/shell)  
   C / Linux 환경에서 명령어 파싱, `fork()`, `execvp()`, history, alias, wildcard, 디렉토리 스택 기능을 구현한 미니 Shell 프로젝트입니다.
