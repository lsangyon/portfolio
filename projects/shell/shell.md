## Execution Result

쉘 프로그램을 실행한 뒤 `pwd`, `ls` 같은 외부 명령어를 실행하고, `pushd`, `dirs`, `history`, `!!` 기능이 정상적으로 동작하는 것을 확인했습니다.

- `pwd`: 현재 디렉토리 출력
- `ls`: 외부 명령어 실행
- `pushd`: 현재 디렉토리를 디렉토리 스택에 저장
- `dirs`: 저장된 디렉토리 스택 출력
- `history`: 실행한 명령어 목록 출력
- `!!`: 직전에 실행한 명령어 재실행
